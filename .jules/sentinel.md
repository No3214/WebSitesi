## 2026-06-19 - XSS in JSON-LD Scripts\n**Vulnerability:** JSON-LD data was being injected directly into <script> tags using dangerouslySetInnerHTML and JSON.stringify, allowing potential XSS if the data contained </script> or other malicious HTML.\n**Learning:** JSON.stringify does not escape HTML characters by default, making it unsafe for direct insertion into HTML contexts.\n**Prevention:** Use a sanitizeJsonLd helper that replaces <, >, &, and ' with unicode escapes when stringifying data for script tags.
